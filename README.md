# 📊 HR Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an end-to-end HR Analytics Dashboard built using **Power BI**, **Power Query**, **DAX**, and **Excel**. The dashboard transforms raw HR data into interactive business insights, enabling HR teams to monitor workforce trends, employee performance, attrition, and training effectiveness.

This project demonstrates the complete data analytics workflow, from data cleaning and modeling to dashboard development and business reporting.

---

## 🎯 Business Objectives

- Monitor workforce size and employee distribution.
- Analyze employee attrition across departments and demographics.
- Evaluate employee performance and employee engagement.
- Measure training effectiveness and training costs.
- Support HR decision-making through interactive dashboards.

## 💡 Key Insights

- Workforce distribution varies across departments.
